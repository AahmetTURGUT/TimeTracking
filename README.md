# Time Tracking
## About
TimeTracker is the only truly free time tracking software. It's a simple time tracker and timesheet app that lets you and your team track work hours across projects. Unlimited users, free forever.


## Requirement
```bash
postgresql =>10
python =>3.6
pip =>18
```
